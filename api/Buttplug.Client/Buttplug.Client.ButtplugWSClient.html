Class ButtplugWSClient

This is the Buttplug C# WebSocket Client implementation. It's intended to abstract away the process of comunicating with a Buttplug Server over WebSockets, so you don't have to worry about the lower level protocol.

Connect(Uri, Boolean)

Creates the connection to the Buttplug Server and performs the protocol hanshake. Once the WebSocket connetion is open, the RequestServerInfo message is sent; the response is used to set up the ping timer loop. The RequestDeviceList message is also sent here so that any devices the server is already connected to are made known to the client.

Important: Ensure that DeviceAdded, DeviceRemoved and ErrorReceived handlers are set before Connect is called.
