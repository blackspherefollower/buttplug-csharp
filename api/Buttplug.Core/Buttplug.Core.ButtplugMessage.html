System.UInt32 aId

The message ID: should be unique within a connection and non-zero, unless its a response to another message or is a server raised event.
System.UInt32 aSchemaVersion

The version of the schema that the message was introduced. Required for cross version schema support.
System.Type aPreviousType

The Type for the previous version of the message, or null. This is used to downgrade messages when cominucating with older clients.
